Project description

Using generative AI to identify and track an object on a video is extremely consuming in terms of processing time and computing power.To make this task more efficient, it is possible to combine a generative AI with an AI discriminative. The discriminative AI allows to detect different objects and track them, whereas generative AI allows to take as input a text describing an object, and to find it among the objects detected.

The project is a research subject with commercial potential, more details techniques cannot be given.
