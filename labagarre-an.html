Project informations

  • Nom: La Bagarre
  • Categorie: Board game
  • Client: -
  • Date: 2022 - 2024
  • Hardskills: Print plotter, laser cutting, graphic design

Description

This project is an unofficial extension of the Risk game with the cards Paris, Lille, Roubaix & Tourcoing, Dunkerque and Bruges. Special cards and new mission cards were added to the base game, as well as a currency system. It is also possible to place forts, the different game modes being explained in a notice.
